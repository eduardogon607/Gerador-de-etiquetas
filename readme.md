# 🏷️ Sistema de Etiquetas Online

Sistema web para geração de etiquetas a partir de planilhas Excel.

## 🌐 Deploy no Vercel

### Pré-requisitos
- Conta no [Vercel](https://vercel.com)
- Conta no [GitHub](https://github.com)

### Passos para Deploy

1. **Criar repositório no GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/seu-usuario/sistema-etiquetas.git
   git push -u origin main
